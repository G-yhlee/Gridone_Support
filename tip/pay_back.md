## 식대 비용 청구 방법

식대 비용 청구 방법 입니다.


1. `비용 청구 문서 다운로드 `를 위해 다음 작업을 수행합니다.

   ```erlang
    Manual Guide of 비용 청구 문서 다운로드

    TASK: [ 비용 서식 폴더 접속 ] - [ 비용 신청서 샘플 (1) 파일 클릭 ]-[ 복사하여 내 로컬 PC에 저장 ] - [ 열기 ]

    INFO:
    비용 서식 폴더 위치: fspub:\01.회사 양식\비용서식
    비용 신청서 샘플 용도 : 차량유지비, 교통비, 식대 비용 등을 청구할때 사용되는 서식 입니다.

   ```

2. `비용 신청서 작성`를 위해 다음 작업을 수행합니다.

   ```erlang
    Manual Guide of 비용 신청서 작성
    
    TASK: [ 비용 신청서 작성 ] - [ 저장 ]
    
    INFO: 
    1. 식대 체크:  □ 박스를 ☑ 박스로 바꿔 넣습니다.
    2. 성명, 사원번호, 부서명, 날짜, 직급을 바꿉니다.
    * 사원번호 확인: [https://ezportal.bizmeka.com 접속] - [근태관리 접속] - [사원번호 확인] 
    3. 영수증 사진을 비용 신청서 작성표 오른쪽에 삽입하여 넣습니다.
    4. 사용금액을 작성합니다.
    5. 하단의 영역에 신청자 이름, 날짜를 작성합니다.
   ```

3. `비용 신청서 메일 보내기`를 위해 다음 작업을 수행합니다.

   ```erlang
    Manual Guide of 비용 신청서 메일 보내기
    
    TASK: [ 메일 접속 ] - [ 비용 신청서 파일 첨부 ] - [ 메일 보내기 ]
    
    INFO: 
    1. 메일 보내기 : all_fin@gridone.co.kr 


   ```
